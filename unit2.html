หน่วยที่ 2

การเขียนโปรแกรมด้วย Scratch (ม.1-ม.2)

สะพานเชื่อมจากการคิดในหัวออกมาเป็น "คำสั่ง" ที่คอมพิวเตอร์เข้าใจได้จริง โดยไม่ต้องกังวลเรื่อง Syntax Error

เนื้อหาหลัก 5 ส่วน

พื้นฐานสำคัญของการเขียนโปรแกรมด้วยบล็อกคำสั่งประกอบด้วย

1

Sequence

โครงสร้างลำดับ

คืออะไร: การวางบล็อกคำสั่งจากบนลงล่างเพื่อให้ตัวละครทำงานตามลำดับ 1, 2, 3

2

Variables

การใช้งานตัวแปร

คืออะไร: การสร้างที่เก็บข้อมูล เช่น คะแนน (Score), เวลา (Timer) หรือพลังชีวิต (HP)

3

Condition

การตัดสินใจและเงื่อนไข

คืออะไร: การใช้บล็อก if-then หรือ if-then-else เพื่อให้โปรแกรมเลือกทำงานตามเงื่อนไข (เช่น ถ้า แตะสีเขียว ให้ ได้คะแนน)

4

Loop

การทำงานวนซ้ำ

คืออะไร: การใช้คำสั่ง repeat หรือ forever เพื่อลดการวางบล็อกซ้ำๆ และทำให้โปรแกรมทำงานต่อเนื่อง

5

Input & Output

การรับข้อมูลและการแสดงผล

คืออะไร: เช่น การกดลูกศรที่คีย์บอร์ดเพื่อเลื่อนตัวละคร หรือการให้ตัวละครพูดตอบโต้

ประโยชน์ที่ได้รับ

✓

เข้าใจตรรกะโปรแกรมเมอร์

ฝึกให้คิดเป็นตรรกะว่า "ถ้าเกิดเหตุการณ์ A จะให้คอมพิวเตอร์ทำ B" ซึ่งเป็นพื้นฐานสำคัญก่อนไปเรียนภาษาที่ยากขึ้นอย่าง Python หรือ Java

✓

ฝึกความคิดสร้างสรรค์ (Creative Thinking)

น้องๆ สามารถสร้างเกม นิทานเคลื่อนไหว หรือสื่อการสอนของตัวเองได้จากจินตนาการ

✓

เรียนรู้ระบบพิกัดและคณิตศาสตร์

การเคลื่อนที่ใน Scratch ช่วยให้เข้าใจเรื่องแกน และเรื่ององศา (ทิศทาง) ได้เห็นภาพชัดเจนขึ้น

✓

ฝึกการแก้ปัญหา (Debugging)

เมื่อโปรแกรมไม่ทำงานตามที่คิด น้องจะได้ฝึกไล่ดูทีละบล็อกว่า "ผิดที่ตรงไหน" ซึ่งเป็นทักษะที่สำคัญมากในโลกการทำงาน
